# MRPZ_Lab_2 — Робота з рядками та випадковими числами
 
## Завдання
 
Сформувати у програмі випадкове ціле число, перетворити його у рядковий літерал та виконати обробку методами класів Java для роботи з рядками.
 
**Варіант 14:** У записі натурального числа видалити першу цифру, а другу цифру збільшити на 1.
 
Наприклад: `64583` → `5583`
 
## Що робить програма
 
1. Генерує випадкове число від 10 до 999999
2. Перетворює його на рядок за допомогою `String.valueOf()`
3. Видаляє першу цифру — `substring(1)`
4. Збільшує другу цифру на 1 — `charAt(0)` + `Character.getNumericValue()`
5. Збирає і виводить результат
## Приклади
 
| Вхід | Результат | Пояснення |
|------|-----------|-----------|
| `64583` | `5583` | видалено `6`, `4` → `5` |
| `31207` | `2207` | видалено `3`, `1` → `2` |
| `99999` | `10999` | видалено `9`, `9` → `10` |
 
## Використані методи рядків
 
- `String.valueOf()` — перетворює число на рядок
- `substring(i)` — повертає рядок починаючи з індексу `i`
- `charAt(i)` — повертає символ за індексом `i`
- `Character.getNumericValue()` — перетворює символ-цифру на число

## Завдання 2
 
Задати довільний текст з кількох речень та виконати його обробку методами класів Java для роботи з рядками.
 
**Варіант 14:** Змінити текст наступним чином — кожне слово має починатися з великої літери.
 
Наприклад: `"java is a language"` → `"Java Is A Language"`
 
### Що робить програма
 
1. Визначає текст як рядковий літерал з кількох речень
2. Виводить текст до обробки
3. Розбиває текст на слова за допомогою `split("\\s+")`
4. Для кожного слова робить першу літеру великою
5. Збирає слова назад через `String.join()`
6. Виводить текст після обробки
### Приклад виконання
 
```
=== Текст ДО ===
java is a powerful language. it was created in 1995.
today it is used everywhere: web, mobile and servers.
 
=== Текст ПІСЛЯ ===
Java Is A Powerful Language. It Was Created In 1995.
Today It Is Used Everywhere: Web, Mobile And Servers.
```
 
### Використані методи
 
- `split("\\s+")` — розбиває рядок на масив слів за пробілами
- `Character.toUpperCase()` — робить символ великим
- `charAt(i)` — повертає символ за індексом
- `substring(i)` — повертає частину рядка починаючи з індексу
- `String.join()` — збирає масив слів назад в один рядок
