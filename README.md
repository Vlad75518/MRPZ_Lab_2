# MRPZ_Lab_2 — Робота з рядками та випадковими числами
 
## Завдання
 
Сформувати у програмі випадкове ціле число, перетворити його у рядковий літерал та виконати обробку методами класів Java для роботи з рядками.
 
**Варіант 14:** У записі натурального числа видалити першу цифру, а другу цифру збільшити на 1.
 
Наприклад: `64583` → `5583`
 
## Що робить програма
 
1. Генерує випадкове число від 10 до 999999
2. Перетворює його на рядок за допомогою `String.valueOf()`
3. Видаляє першу цифру — `substring(1)`
4. Збільшує другу цифру на 1 — `charAt(0)` + `Character.getNumericValue()`
5. Збирає і виводить результат
## Приклади
 
| Вхід | Результат | Пояснення |
|------|-----------|-----------|
| `64583` | `5583` | видалено `6`, `4` → `5` |
| `31207` | `2207` | видалено `3`, `1` → `2` |
| `99999` | `10999` | видалено `9`, `9` → `10` |
 
## Використані методи рядків
 
- `String.valueOf()` — перетворює число на рядок
- `substring(i)` — повертає рядок починаючи з індексу `i`
- `charAt(i)` — повертає символ за індексом `i`
- `Character.getNumericValue()` — перетворює символ-цифру на число
